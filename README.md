# library_seat

건국대학교 도서관 좌석 모니터링/예약 보조 GUI 스크립트입니다.

## Requirements

- Python 3.10+
- Chrome 브라우저

## Run

```bash
python library.py
```

## Optional Environment Variables

자동 로그인을 사용하려면 아래 환경 변수를 설정합니다.

- `KONKUK_LIBRARY_ID`
- `KONKUK_LIBRARY_PW`

## Notes

- 빌드 산출물(`build/`, `dist/`)과 가상환경(`.venv/`)은 Git에 포함하지 않습니다.
